## 📌 Overview
The **H. pylori Detector** is an AI-powered tool designed to assist pathologists by automatically identifying *Helicobacter pylori* bacteria in whole slide images (WSIs) of gastric tissue.

It combines deep learning with an interactive visualization interface for efficient and accurate detection.

---

## 🚀 Features

- 🔬 Automated detection of *H. pylori* in histopathology slides  
- 🧠 Vision Transformer-based deep learning model  
- 🎯 Handles extremely sparse object detection  
- 🎛️ Interactive confidence threshold slider  
- 🖼️ Whole Slide Image (WSI) support  
- ⚡ Fast inference with optimized architecture  
- 🧩 Multi-task learning (Detection + Super-Resolution)
